## 프로젝트 소개

- 플랫폼 : 안드로이드/모바일
- 제작 인원 : 7 인 (팀 구성 ― 안드로이드 1명, iOS 1명 백엔드 4명, 디자이너 1명)
- 제작 기간 : 49 일 (2024.07.12 ~ 2024.08.30)
- 사용 기술 : Kotlin, XML, ViewBinding, Glide, Retrofit2, ViewModel, FCM, Google Map, Kakao OAuth, Google OAuth, Kakao Map 

'ADEGO'는 약속 시간에 자주 늦는 친구들의 지각 문제를 해결하기 위해 개발된 조금 새로운 방식의 지각 방지 앱으로<br>
기존의 친구들과 일상 속 시간 약속에서 겪던 불편함을 개선하고, 사용자 경험을 극대화한 약속 관리 서비스입니다.<br>
기존의 방식이 벌금 걷기, 약속 시간을 조정하는 등의 번거로운 과정을 거쳤다면, ADEGO는 친구들과 약속 시간과 장소를 설정한 후,<br>
딥링크를 통해 간편히 친구들을 초대하고 약속 시간이 다가올 시 자동으로 위치 공유와 알림 발송이 가능하게 하는 기능을 통해<br>
자주 지각하는 친구들의 위치와 수면등의 여부를 몰라 답답하던 이러한 불편함을 해소하였습니다.

    - 구글 OAuth

    - 약속 날짜

    - 약속 주소
    - 딥링크 약속 초대 링크
- 설정
    - 로그아웃 기능
    - 회원탈퇴 기능

## 느낀 점

어데고 프로젝트에서 Kotlin과 XML을 사용한 Android 앱 개발 총괄 및 구글 플레이 스토어 출시를 담당하였으며 
개발한 주요 기능으로는 포그라운드 위치 전송 서비스, 딥링크 구현이 있습니다. 해당 앱을 통해 처음으로 포그라운드 서비스 및 Goolge OAuth와
위치 관련 기능, 딥링크 등 새로운 기술 사용 및 습득 경험과 다양한 사람들과 하는 팀 프로젝트 경험을 얻을 수 있었습니다. 
또한 해당 프로젝트를 잘맞는 개발 관련 친구들을 알게 되었고 서개모라는 커뮤니티 겸 팀을 만들게 되었습니다. 
이외에 아쉬운 점과 느낀 점이 많지만 나머지는 블로그에 작성하겠습니다
